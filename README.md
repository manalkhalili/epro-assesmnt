# EPRO IPO Readiness Assessment Tool

A clean and professional WordPress plugin that helps companies evaluate their readiness for **Initial Public Offering (IPO)**.

### Features
- Bilingual support: **Arabic (RTL)** + **English**
- Multi-step assessment form (financial data + governance + IPO criteria)
- Automatic scoring system:
  - Financial Health
  - Governance
  - IPO Readiness
  - Overall Score (out of 100)
- Smart IPO Blocker detection
- Beautiful, compact printable PDF reports (optimized for A4)
- Professional email reports sent automatically
- Admin dashboard with results overview and CSV export

### Perfect For
- Investment banks
- Financial advisory firms
- IPO consultants
- Companies preparing for listing

### Installation
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin
3. Use shortcode `[epro_assessment]` for Arabic or `[epro_assessment_en]` for English
